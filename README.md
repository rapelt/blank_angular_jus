
## Installation

Global dependencies

```
    npm install -g gulp bower
```

Clone project

```
    git clone https://github.com/epenkman/oss-cafe.git
    cd oss-cafe
    npm install
```

## Build and Run

Local developer preview:
```
    gulp serve
```

Unit tests with karma.js
```
    gulp test
```

Generate deployable artefact
```
    gulp build
```
